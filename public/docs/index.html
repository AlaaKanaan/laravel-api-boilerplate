Introduction

Base URL: https://laravel-boilerplate-api.test

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Endpoints

Handle an incoming registration request.

Example request:
curl --request POST \
    "https://laravel-boilerplate-api.test/api/v1/auth/sign-up" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"zad\",
    \"email\": \"cherzog@example.net\",
    \"password\": \"nulla\"
}"

const url = new URL(
    "https://laravel-boilerplate-api.test/api/v1/auth/sign-up"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "zad",
    "email": "cherzog@example.net",
    "password": "nulla"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Handles the password reset process for a user.

Validates the provided request data including OTP, email, and password. It checks the cached OTP, updates the user's password if validation succeeds, and removes the OTP identifier value after completion.

Example request:
curl --request POST \
    "https://laravel-boilerplate-api.test/api/v1/auth/reset-password" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"otp\": \"ipsam\",
    \"email\": \"mohammed.becker@example.net\",
    \"password\": \"corrupti\"
}"
